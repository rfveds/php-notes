### Nginx in PHP
- Nginx is a web server that can be used as a reverse proxy server.
- Nginx can be used to serve static files and PHP files.
- Nginx is known for its high performance and low memory usage.
- Nginx is often used in combination with PHP-FPM.

### PHP-FPM
- PHP-FPM (FastCGI Process Manager) is an alternative PHP FastCGI implementation.
- PHP-FPM is used to handle PHP requests.
- PHP-FPM is known for its high performance and low memory usage.
- PHP-FPM is often used in combination with Nginx.

#### Reverse proxy server
- A reverse proxy server is a server that sits between clients and backend servers.
- The reverse proxy server forwards client requests to the backend servers.
